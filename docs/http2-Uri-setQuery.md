# [HTTP Message v2](http2.md) / [Uri](http2-Uri.md) :: setQuery
 > im\http2\msg\Uri
____

## Description
Set a querystring key part

## Synopsis
```php
setQuery(string $name, null|string $value): void
```

## Parameters
| Name | Description |
| :--- | :---------- |
| name | The key to set |
| value | The value to set.<br />You can set this to `NULL` to remove it. |
