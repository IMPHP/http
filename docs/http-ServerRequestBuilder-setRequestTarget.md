## Description
Set a new request target

Note that this will stop the request target from being updated
when changing the Uri object

## Synopsis
```php
public setRequestTarget(string $requestTarget): void
```

## Parameters
| Name | Description |
| :--- | :---------- |
| requestTarget | The new request target |
